Creating new controller (simulated .ino file)

1. create a copy of program_template directory

2. rename the copied directory

3. add the newly created directory to CmakeLists.txt file located at the same folder as this README

4. inside the created directory, go to the CMakeLists.txt file and replace "program_template" with the name of the created directory at line 4

5. add your code to krembot_main.h file the same way as you do with .ino

6. compile argos3-user directory
